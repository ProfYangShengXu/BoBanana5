# 🍌 Bobanana 5.0 — 三分钟上手

你只需要会两件事：
1. **打字**（在 Reasonix 对话框输入指令）
2. **点鼠标**（双击图标，点按钮）

---

## 🚀 方法一：双击就开干（最简单）

```bash
# Windows: 双击 bobanana.bat
# 弹出一个菜单，选 1 输入目标就行
```

或者：

```bash
python bobanana.py
```

## 💬 方法二：一句话

```bash
# 在 Reasonix 对话框输入：
python bobanana.py "帮我做一个 Todo 应用"

# 然后去喝杯咖啡，回来就好了
```

## 📊 方法三：看仪表盘

```bash
python bobanana.py --gui
```

## 🛠️ 第一次使用

```bash
# 一键安装全部依赖
python bobanana.py --setup
```

---

## 就这么简单

| 你想干嘛 | 输入 |
|---------|------|
| 开干 | `python bobanana.py "你的目标"` |
| 看仪表盘 | `python bobanana.py --gui` |
| 看角色卡 | `python bobanana.py --cards` |
| 跑测试 | `python bobanana.py --test` |
| 双击启动 | 点 `bobanana.bat` |
| 第一次安装 | `python bobanana.py --setup` |

**不用记任何命令。不用装任何东西。双击就能用。**
