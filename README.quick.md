# 🍌 Bobanana 5.0 — 三分钟上手

你只需要会两件事：
1. **打 `/bb <目标>`**（在 Reasonix 对话框输入）
2. **点鼠标**（双击图标，点按钮）

---

## 🚀 方法一：在 Reasonix 用 `/bb`（推荐）

```text
/bb 帮我做一个记账软件
```

AI 自动推进：设计 → 开发 → 审查 → CL 终审

## 🖥️ 方法二：看仪表盘

```bash
python bobanana.py --gui
```

## 🛠️ 第一次使用

```bash
# Reasonix 桌面端：设置 → 插件管理 → 选择 bobanana-plugin 文件夹
```

---

## 就这么简单

| 你想干嘛 | 输入 |
|---------|------|
| 启动管线 | `/bb 你的目标` |
| 看仪表盘 | `python bobanana.py --gui` |
| 跑测试 | `python -m pytest tests/ -v` |
| 讨论需求 | `/docs 项目: xxx` |
