🎨 Bobanana 5.0 GUI — UI/UX 技术方案

基于 EUI-NEO 框架 (C++17/OpenGL) 的桌面仪表盘 UI/UX 升级

1. 配色系统

1.1 架构

定义 Theme 结构体，包含所有 UI 元素的颜色属性。通过 ThemeManager 在运行时切换。

struct Theme {
    std::string name;
    eui::Color background;     // 背景色
    eui::Color surface;        // 卡片/面板背景
    eui::Color text_primary;   // 主文字
    eui::Color text_secondary; // 副文字
    eui::Color accent;         // 强调色
    eui::Color success;        // 绿色
    eui::Color danger;         // 红色
    eui::Color warning;        // 黄色
    eui::Color info;           // 蓝色
    eui::Color border;         // 边框色
    // 渐变
    eui::Color gradient_start, gradient_end;
};

2. 动画系统

2.1 动画引擎

利用 EUI-NEO 的帧渲染循环，在 compose() 中计算动画插值。

// 动画状态
struct AnimState {
    float progress;           // 0.0 → 1.0
    float speed;              // 每秒进度增量
    bool active;
    std::string easing;       // "ease-out" | "linear" | "bounce"
    std::chrono::steady_clock::time_point last_time;
};

2.2 动画类型

类型目标时长效果
Fade In面板加载0.3s透明度 0→1
Slide Up状态更新0.4sY 轴偏移 20→0

4. 视觉效果

  • 渐变背景 — 标题栏和面板使用线性渐变而非纯色
  • 发光边框 — 当前活跃面板有霓虹光晕边框
  • 毛玻璃 — 面板背景带模糊效果 (backdrop-filter 模拟)
  • 节点连线动画 — 状态机边用虚线动画流动
  • 评分动画 — CL 分数以数字滚动动画呈现
  • 粒子背景 — 背景飘浮缓慢移动的细小光点
